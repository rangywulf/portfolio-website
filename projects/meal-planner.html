MEAL PLANNER SHEETS + APPS SCRIPT

DateDecember 2025
TypePersonal Project
PlatformGoogle Sheets
Overview / Goal

A lightweight meal planning system built entirely in Google Sheets and Apps Script that generates four weeks of meals while preventing repetitive dinners. Built with no external dependencies.

The tool focuses on planning and decision-making over recipe management, deliberately separating the "what am I eating?" question from "how do I make it?" keeping it fast and frictionless to use.

Key Features
  • Automated meal generation across four full weeks, with anti-repetition logic to keep dinners varied
  • Customizable dropdowns for quick swaps without regenerating the whole plan
  • Consolidated calendar view so the full month is visible at a glance
  • Optional side dish pairing built into the generation logic
  • No external dependencies, runs entirely inside Google Sheets via Google Apps Script
